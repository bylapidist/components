# Box

> The `Box` component is a foundational building block for creating other components in your application. It provides 
a simple way to control gutters and elevation, ensuring that your designs remain consistent and aligned with the theme.
By using `Box` as a wrapping element for compound components, you can manage their padding and shadows without needing 
to define these properties repeatedly.

## Basic Usage

To use the `Box` component, simply import it into your file and use it as a wrapping element around other components. 
Here's an example that will render a `div` by default:

<Source dark code={`
import { Box } from '@lapidist/components';

function App() {
    return (
        <Box>Hello, World!</Box>
    );
}
`} />

And here's a full example which shows all valid props:

<Source dark />

## Controlling gutters

To control the padding of the Box, you can use the `gutter`, `gutterX`, and `gutterY` props. These props accept one of 
four available gutters: `none`, `small`, `medium`, and `large`. Here's an example of how to use the `gutter` prop:

<Source dark code={`
import { Box } from '@lapidist/components';

function App() {
    return (
        <Box gutter="small">Hello, World!</Box>
    );
}
`} />

This will add a small padding to the `Box` like this example:

## Controlling elevation

To control the shadow of the Box, you can use the elevation prop. This prop accepts one of four available elevations: 
`none`, `small`, `medium`, and `large`. Here's an example of how to use the `elevation` prop:

<Source dark code={`
import { Box } from '@lapidist/components';

function App() {
  return (
      <Box elevation="medium">Hello, World!</Box>
  );
}
`} />
