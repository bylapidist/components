# Link
> A primitive for hyperlinks. They should be used when you want to navigate to a new page, or to an anchor on the current
page. They should not be used to invoke actions (e.g. submitting a form or toggling an option) - use a 
[Button](?path=/docs/inputs-button--documentation) in these scenarios instead.

## Controlling font size
The `size` prop controls font size. There are three available font sizes, defined as `small`, `medium` and `large`.

### Small
> The `small` size sets the font size to `--font-size__200`
