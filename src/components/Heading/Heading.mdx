# Heading

> A primitive for controlling headings. Use headings to establish a consistent approach to typography hierarchy,
> ensuring they are aligned with the theme. It builds upon the [Text](?path=/docs/typography-text--documentation) component.

<Source dark />

## Controlling heading size

The `size` prop controls the heading level. There are three available sizes, defined as `small`, `medium`, and `large`.
You can change the semantic element using the `as` prop if you wish to use the visual appearance of a different level.
By default `large` maps to `h1`, `medium` to `h2` and `small` to `h3`.

> The `large` size should be used for the primary heading on a page or section.
