# Button
> A primitive for invoking actions. They should not be used to navigate to new pages or anchors on the current page - use 
a [Link](?path=/docs/typography-link--documentation) in these scenarios instead.

## Example
<Source dark />

## Controlling kind
The `kind` prop controls button appearance. There are four available kinds, defined as `primary`, `secondary`,
`tertiary` and `danger`.

### Primary
> The `primary` kind should be used as the main call to action within a page or button group.
<Canvas>
    <Story of={ButtonStories.Primary} />
</Canvas>

### Secondary
> The `secondary` kind should be used as a secondary call to action within a page or button group.
<Canvas>
    <Story of={ButtonStories.Secondary} />
</Canvas>

### Tertiary
> The `tertiary` kind should be used as any button that is not a call to action or destructive.
<Canvas>
    <Story of={ButtonStories.Tertiary} />
</Canvas>

### Danger
> The `danger` kind should be used for triggering any destructive action.
<Canvas>
    <Story of={ButtonStories.Danger} />
</Canvas>

## Controlling size
The `variant` prop controls button size. There are three available sizes, defined as `small`, `medium` and `large`.
