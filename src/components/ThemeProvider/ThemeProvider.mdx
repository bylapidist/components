# ThemeProvider

> A wrapper for providing theming support to all children.

## Controlling theme

The `theme` prop controls the theme used by all child components. There are two available themes, defined as `light` and `dark`.

### Light

> Use the `light` theme by default, and for users who prefer light themes.

> Use the `dark` theme for users who prefer dark mode.
