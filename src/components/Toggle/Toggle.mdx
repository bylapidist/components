# Toggle

> A primitive for toggling between two states. They are functionally an individual checkbox under the hood, but styled more like
> a [Button](?path=/docs/inputs-toggle--documentation).

## Example

<Source dark />

## Controlling level

The `level` prop controls the toggle appearance. There are four available levels, defined as `primary`, `secondary`,
`tertiary` and `danger`.

### Primary

> The `primary` level should be used as the main call to action within a page or toggle group.

<Canvas>
    <Story of={ToggleStories.PrimaryMedium} />
</Canvas>

### Secondary

> The `secondary` level should be used as a secondary call to action within a page or toggle group.

<Canvas>
    <Story of={ToggleStories.SecondaryMedium} />
</Canvas>

### Tertiary

> The `tertiary` level should be used as any toggle that is not a call to action or destructive.

<Canvas>
    <Story of={ToggleStories.TertiaryMedium} />
</Canvas>

### Danger

> The `danger` level should be used for toggling any destructive action.

<Canvas>
    <Story of={ToggleStories.DangerMedium} />
</Canvas>

## Controlling size

The `size` prop controls toggle size. There are three available sizes, defined as `small`, `medium` and `large`.

### Small

> Use the small size for toolbars or where space is constrained.
