# Tag

> A primitive for displaying short inline status messages or information.

## Example

<Source dark />

## Controlling level

The `level` prop controls the tag appearance. There are four available levels, defined as `primary`, `secondary`,
`tertiary` and `danger`.

### Primary

> The `primary` level should be used as the main or affirmative status.

<Canvas>
    <Story of={TagStories.Primary} />
</Canvas>

### Secondary

> The `secondary` level should be used as a secondary status when there is already a primary status.

<Canvas>
    <Story of={TagStories.Secondary} />
</Canvas>

### Tertiary

> The `tertiary` level should be used as any status message that is not affirmative or destructive, or when there's neither primary nor secondary status.

<Canvas>
    <Story of={TagStories.Tertiary} />
</Canvas>

### Danger

> The `danger` level should be used for any destructive or error status.

<Canvas>
    <Story of={TagStories.Danger} />
</Canvas>
